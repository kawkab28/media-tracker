# Media Tracker Application

A simple web application to manage a personal media collection (movies, music, games).

## Features

- Add, edit, and delete media items
- Search media by title, creator, genre, status, or release date
- User authentication (register/login)
- AI-powered media recommendations
- Simple web interface

## Tech Stack

Backend:
- Node.js
- Express

Database:
- SQLite

Frontend:
- HTML
- Bootstrap
- JavaScript

AI Feature:
- Media recommendation endpoint

```bash
npm install
npm run dev
